REFLECTION

The WEB222 course provided a broad understanding of web development. The parallel study of html, javascript and css made it possible to see the result of my work immediately and try different concepts of web development at the same time. Unlike OOP, where it is often quite difficult to at least identify the problem, the web is more high-level and easier for a beginner to "understand". Debugging allows me to receive feedback not only in the development environment window (for example, in the VS code), but also in the browser through development tools. Through the flexboxdefense game, I learned the concept of a flexbox model for distributing space between elements in an interface. I can't imagine games where one could study as clearly, for example, C ++ constructors. I tried working with different positioning objects (static, relative, fixed, absolute, sticky) and css selectors as well.

Web development is an integral part of full stack development. This semester I have been writing C++ code for a simple Oracle database interface. The results of queries and menus were displayed in a black output window. It looked pretty dull. I believe that in the next courses the black screen will be replaced by a more pleasant web interface, and thus, all three technologies complementing each other, will allow me to create an almost complete final product.
